# Interactive Form Validation

A modern, responsive, and accessible registration form built as a front-end internship task. It features real-time validation, dynamic error handling, and a password strength meter, all built without the use of external CSS or JavaScript frameworks.

## 🚀 Features

* **Real-Time Validation**: Instant feedback provided to the user as they type.
* **Regex Pattern Matching**: Ensures emails and phone numbers follow correct syntax.
* **Password Strength Meter**: Dynamically calculates and displays password strength (Weak, Medium, Strong).
* **Accessible Design**: Uses semantic HTML5, `aria-live` for screen reader error announcements, and proper label associations.
* **Responsive Layout**: Mobile-first design that scales beautifully to tablet and desktop.
* **Success Feedback**: Animated toast notification upon successful validation and submission.

## 🛠️ Technologies Used

* **HTML5** (Semantic structure)
* **CSS3** (Flexbox layout, CSS Variables, Transitions)
* **Vanilla JavaScript (ES6)** (DOM manipulation, event handling, Regex)
* **Google Fonts** (Poppins)
* **FontAwesome** (Icons)
