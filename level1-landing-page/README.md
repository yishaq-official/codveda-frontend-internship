# YishaqTech Landing Page

A modern, professional, and fully responsive landing page designed for a technology company. Built completely from scratch without external CSS or JS frameworks to demonstrate clean code practices.

## 🚀 Features

- **Fully Responsive Design:** Follows a mobile-first approach, adapting beautifully from mobile screens up to large desktops.
- **CSS Grid & Flexbox:** Uses modern layout models for precise structural control.
- **Custom Animations:** - Intersection Observer API for high-performance scroll-reveal animations.
  - Hover effects on feature cards, service boxes, and buttons.
  - Dynamic sticky navigation that changes style on scroll.
- **Mobile Navigation:** Custom hamburger menu implementation.
- **Accessibility & SEO:** Built with semantic HTML5 tags and descriptive alt texts.

## 🛠️ Technologies Used

- **HTML5:** Semantic markup.
- **CSS3:** Custom variables, CSS Grid, Flexbox, media queries, and animations.
- **Vanilla JavaScript (ES6):** DOM manipulation, event listeners, and Intersection Observers.
- **Google Fonts:** Utilizing 'Poppins' for clean, modern typography.
- **FontAwesome:** Scalable vector icons.
